# Customer-Purchase-Behavior-Analysis

## Project Overview
This project focuses on predicting online retail revenue based on customer purchasing behavior.

## Business Goal
Improve pricing and inventory decisions using data-driven insights.

## Tools & Technologies
- Python
- Pandas
- Scikit-learn
- Data Visualization

## Models Used
- Linear Regression
- Random Forest Regression

## Results
- Random Forest achieved R² = 0.84
- Evaluated using RMSE
- KPI: Predictions within ±15% of actual revenue

## Key Insights
- Lower-priced products drive higher purchase quantities
